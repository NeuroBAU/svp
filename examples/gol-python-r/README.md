# Example: gol-python-r

This directory contains the SVP example project for `gol-python-r`.

Bundled artifacts will include:
- `project_context.md` — project context
- `stakeholder_spec.md` — stakeholder specification
- `blueprint_prose.md` — blueprint tier 1 descriptions
- `blueprint_contracts.md` — blueprint tier 2/3 contracts
- `oracle_manifest.json` — oracle trajectory configuration

This example is bundled with SVP 2.2 to demonstrate the pipeline output format.
