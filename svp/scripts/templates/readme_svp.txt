===============================================================================
                          SVP-MANAGED PROJECT
===============================================================================

This project is managed by the Stratified Verification Pipeline (SVP).

DO NOT manually edit the following files:
  - pipeline_state.json
  - .svp/ directory contents
  - CLAUDE.md (regenerated by SVP)

These files are maintained by the SVP pipeline and manual edits may corrupt
the pipeline state or cause unexpected behavior.

To interact with the pipeline, use the SVP slash commands:
  /svp:status  - View current pipeline status
  /svp:save    - Save pipeline state
  /svp:quit    - Exit the pipeline gracefully
  /svp:help    - Get help with the current stage
  /svp:hint    - Get a hint for the current task
  /svp:ref     - Index reference documents (Stages 0-2)
  /svp:redo    - Report an error for reclassification (Stages 2-4)
  /svp:bug     - Report a post-delivery bug (after Stage 5)
  /svp:clean   - Clean up workspace artifacts

For more information about SVP, see the project documentation.
