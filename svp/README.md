# SVP Plugin

This directory contains the SVP Claude Code plugin. See the repository root `README.md` for full documentation.

The plugin components:

- `agents/` — Agent definition files (AGENT.md format)
- `commands/` — Slash command files (command .md format)
- `hooks/` — Write authorization hooks and shell scripts
- `scripts/` — Deterministic pipeline scripts (Python)
- `skills/orchestration/` — Orchestration skill (SKILL.md)
